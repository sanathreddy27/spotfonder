# 🌍 SpotFonder

SpotFonder is an AI-powered travel planner that helps users explore tourist destinations across India. It provides personalized travel itineraries, weather updates, hotel and restaurant recommendations, attractions, budget planning, and Google Maps integration.

---

## ✨ Features

- 🤖 AI-powered 3-Day & 5-Day travel itinerary
- 🌤 Live weather information
- 🏨 Hotel recommendations
- 🍽 Restaurant recommendations
- 📍 Tourist attractions
- 💰 Budget planner
- 🗺 Google Maps integration
- 🔐 User Authentication (JWT)
- ❤️ Save favorite trips
- 📱 Responsive design

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router
- Axios
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication

### APIs
- Geoapify API
- OpenWeather API
- Pexels API

---

## 🚀 Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
